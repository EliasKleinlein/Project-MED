# Flucht aus der Python-Prüfung

Ein kleines Text-Adventure in Python.

Der Spieler ist in einer schwierigen Python-Prüfung gefangen und muss verschiedene Räume, Rätsel und Programmieraufgaben lösen, um zu entkommen.

#  Spielidee

Du wachst in einem Prüfungsraum auf. Die Uhr tickt.

Vor dir befinden sich mehrere Türen. Hinter jeder Tür warten neue Herausforderungen:

- Python-Fragen
- Rätsel
- Fallen
- Gegenstände zum Sammeln
- Geheimräume
- Der Endgegner: Professor SyntaxError

Nur wer die Aufgaben richtig löst, kann aus der Prüfung entkommen.
